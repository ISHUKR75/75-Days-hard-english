## 🚀 **Getting Started**

### **Prerequisites**
- Node.js 18+ or Bun
- npm, yarn, pnpm, or bun

### **Installation**

```bash
# Clone the repository
git clone https://github.com/yourusername/75-days-hard-english.git

# Navigate to the project
cd 75-days-hard-english

# Install dependencies
npm install
# or
yarn install
# or
pnpm install
# or
bun install

# Run the development server
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:5000](http://localhost:5000) in your browser.

## 🎯 **Key Features Explained**

### **1. Learning System**

Every topic follows the **EXACT same structure**:

1. **Explanation** - Complete theory in Hindi & English
2. **Concept** - Core concept with visual aids
3. **Real-Life Examples** - Daily conversation scenarios
4. **Professional Examples** - Office & business usage
5. **Common Mistakes** - Error detection & correction
6. **Speaking Tips** - Pronunciation & fluency tips
7. **Memory Tricks** - Easy-to-remember techniques
8. **Vocabulary** - 500-1000 new words per topic
9. **Verbs** - All forms (V1, V2, V3, V4, V5)
10. **Sentence Patterns** - Structure templates
11. **Daily Conversation** - Everyday dialogues
12. **Office Conversation** - Professional scenarios
13. **Practice Questions** - 500-1000 Hindi → English
14. **Test Questions** - 100-200 MCQs/tests
15. **Speaking Test** - Roleplay & pronunciation
16. **Writing Test** - Essay & email practice
17. **Listening Test** - Audio comprehension
18. **Revision** - Summary & quick review
19. **Story** - Story using only learned grammar
20. **Essay** - Essay with learned concepts
21. **Dialogue** - Conversational practice
22. **Summary** - Quick recap
23. **Progress** - Tracking & analytics
