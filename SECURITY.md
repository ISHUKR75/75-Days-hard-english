# Security Policy

## Supported Versions
Only the latest version of the 75 Days Hard English Course platform is actively supported with security updates.

| Version | Supported |
| ------- | --------- |
| 1.0.x   | Yes       |
| < 1.0   | No        |

## Reporting a Vulnerability
If you discover a security vulnerability in this project, please do not open a public issue. Instead, email us directly at support@example.com (or your designated open source email).

We will investigate the report and respond with our findings and a resolution plan as quickly as possible.
